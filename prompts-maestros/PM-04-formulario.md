# PM-04 — Formulario funcional con Formspree

> Bloque 2 — Hacer el formulario realmente funcional.

```
Toma el formulario de contacto del index.html. Hazlo funcional con Formspree.

1. Cambia action a "https://formspree.io/f/REEMPLAZAR_AQUI"
2. Agrega method="POST"
3. Validación con JavaScript:
   - Mostrar error en rojo bajo cada campo si está vacío al enviar
   - Validar formato de email
   - Validar que el teléfono tenga 9 dígitos
4. Al enviar correctamente: mensaje verde "¡Mensaje enviado!" y limpiar
   formulario.

Devuelve solo el HTML del formulario y el <script> con la lógica.
```
