# PM-01 — Landing Page completa

> Bloque 2 — Generación inicial del proyecto.

```
Actúa como desarrollador frontend senior con experiencia en marketing digital.

Crea una landing page de una sola página (one-page) en un único archivo
index.html para el curso "Programando con Claude IA" de OJB Soluciones
Informáticas.

Stack:
- HTML5 semántico
- Tailwind CSS via CDN (https://cdn.tailwindcss.com)
- JS vanilla mínimo si hace falta

Paleta:
- Naranja primario: #E8742C
- Azul oscuro: #1F3A5F
- Blanco de fondo
- Grises Tailwind para textos secundarios

Estructura en este orden exacto:
1. Navbar fija con logo "OJB" y menú (Inicio, Beneficios, Temario, Testimonios,
   Contacto). Botón naranja "Inscríbete" al final.
2. Hero a pantalla completa: título grande "Aprende Programación con
   Inteligencia Artificial", subtítulo, dos botones (CTA naranja "Quiero
   inscribirme" y secundario "Ver temario"), fondo con degradado sutil.
3. Sección de 3 beneficios en grid: "Clases prácticas", "Proyectos reales",
   "Certificado incluido". Cada uno con emoji como ícono y párrafo corto.
4. Sección Temario con 6 módulos numerados:
   1) Introducción a Claude IA
   2) Prompts efectivos
   3) Skills de Claude
   4) Proyecto: Landing Page
   5) Proyecto: Tienda con MySQL
   6) MCP y siguientes pasos
5. Sección Testimonios con 3 tarjetas (avatar circular con iniciales, nombre,
   cargo, comentario).
6. Sección Contacto con formulario (nombre, email, teléfono, mensaje, botón
   naranja "Enviar mensaje") y datos de contacto: "WhatsApp: 970 779 871"
   y "Lima, Perú".
7. Footer oscuro con copyright "© 2026 OJB Soluciones Informáticas".

Requisitos técnicos:
- Totalmente responsive (móvil, tablet, desktop)
- Animaciones suaves al hacer scroll
- Smooth scroll en los enlaces del navbar
- Validación HTML5 en el formulario
- Accesibilidad básica: alt en imágenes, aria-labels en botones

Devuelve únicamente el código completo del archivo index.html, sin
explicaciones ni comentarios extra.
```
