# PM-09 — Conectar frontend con backend

> Bloque 4 — Que el catálogo cargue datos reales.

```
Modifica catalogo.html para que:

1. Al cargar la página, llame con fetch() a productos.php
2. Pinte dinámicamente las tarjetas con los datos reales
3. Si la respuesta falla, muestre "No se pudieron cargar los productos"
   en un banner rojo
4. Mientras carga, muestre un spinner centrado

Devuelve solo:
- El HTML del contenedor donde van las tarjetas (con id="catalogo")
- El bloque <script> con la lógica fetch

No repitas el resto del HTML.
```
