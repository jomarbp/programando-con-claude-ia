# PM-07 — Catálogo frontend

> Bloque 4 — Generar la interfaz del catálogo.

```
Crea el archivo catalogo.html para una tienda online.

Stack:
- HTML + Tailwind CSS (CDN)

Diseño:
- Navbar superior con logo y ícono de carrito (con contador)
- Grilla responsive de tarjetas:
  - 4 columnas en desktop
  - 2 en tablet
  - 1 en móvil
- Cada tarjeta debe tener:
  - Imagen del producto
  - Nombre
  - Precio en formato "S/ 0.00"
  - Descripción corta (2 líneas máx)
  - Botón "Agregar al carrito"

Colores:
- Naranja #E8742C como acento
- Blanco de fondo

Por ahora deja los productos hardcodeados (8 productos de ejemplo).
Luego los conectaremos a la BD.

Devuelve solo el código del archivo catalogo.html.
```
