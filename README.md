## 📋 Sobre el curso

Aprende a programar proyectos reales (landing pages, tiendas online, conexión con bases de datos) usando **Claude IA** en su versión gratuita. Domina prompts, skills y MCP en una sola sesión de 3 horas.

| Datos rápidos | |
|---|---|
| ⏱ **Duración** | 3 horas |
| 👥 **Audiencia** | Mixta (principiantes a profesionales) |
| 💻 **Herramienta** | Claude Code (web + desktop) |
| 💰 **Versión** | Gratuita |
| 🎯 **Objetivo** | Crear 2 proyectos reales funcionales |

## ⚡ Inicio rápido (3 pasos)

### 1. Clonar este repo
```bash
git clone https://github.com/jomarbp/programando-con-claude-ia.git
cd programando-con-claude-ia
```

### 2. Instalar skills oficiales de Claude
```powershell
# Windows PowerShell
.\scripts\instalar-skills.ps1
```

```bash
# Linux/Mac
bash scripts/instalar-skills.sh
```

### 3. Empezar con el Bloque 0
```bash
cd bloque-0-setup
# Sigue las instrucciones del README.md
```
